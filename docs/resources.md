# 📚 NGCP UAV Software Resources (2025–26)

This document collects beginner-friendly resources for our UAV Software Team to build skills in the tech stack.

---

## 🌱 Core Skills
- **Ubuntu 24.04 Basics**
  - [Ubuntu Official Tutorials](https://ubuntu.com/tutorials)  
  - [Linux for Programmers – YouTube](https://www.youtube.com/playlist?list=PLS1QulWo1RIb9WVQGJ_vh-RQusbZgO_As)  
  - Learn: terminal navigation, package management (`apt`), file permissions  

- **Git & GitHub (version control)**
  - [Pro Git Book (free)](https://git-scm.com/book/en/v2)  
  - [GitHub Hello World guide](https://docs.github.com/en/get-started/start-your-journey/hello-world)  

---

## ✈️ UAV Development
- **PX4 Autopilot**
  - [PX4 User Guide](https://docs.px4.io/main/en/)  
  - [PX4 YouTube Channel](https://www.youtube.com/@px4autopilot)  
  - Focus: SITL (software-in-the-loop), flight modes, missions  

- **MAVSDK (Python/C++)**
  - [MAVSDK Python Guide](https://mavsdk.mavlink.io/main/en/python/)  
  - [MAVSDK C++ Guide](https://mavsdk.mavlink.io/main/en/cpp/)  
  - Start with Python (easier), then explore C++  

- **Gazebo Simulator**
  - [Gazebo Tutorials](https://classic.gazebosim.org/tutorials)  
  - Learn: spawning UAVs, simulating planes, integrating with PX4 SITL
